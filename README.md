ROFLBALT
========

A Canabalt-inspired sidescroller in ASCII (with ANSI color!) for your console.

Run it in Terminal.app or iTerm.app with xterm-256color.

Instructions
------------

With Ruby 1.9 installed, run:

    cd roflbalt
    ./rofl
